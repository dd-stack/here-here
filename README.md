# 여기 여기 붙어라

"여기 여기 붙어라"는 가볍게 주고 받을 수 있는 캐주얼 초대장 서비스입니다.<br>
초대장 만들기 및 공유, 일정 등록, 보낸 초대장 & 받은 초대장 관리 등을 할 수 있습니다.<br>
